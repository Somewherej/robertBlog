# roberBlog

基于Springboot的前后端分离博客系统

# 技术选型

- **后端技术栈**
  - springboot
  - mybaits
  - jwt

- **数据库**
  - mysql
- **前端技术**
  - vue
  - element-ui
- **中间件技术**
  - redis
- **运维技术**
  - centos
  - docker
  - nginx
  - maven

#  项目结构

```c++
│  Deployment Descriptor.md      //部署文档
│  LICENSE.txt   
│  README.md 
│  images
│   
├─前端
│  ├─admin-vue                  //前台代码         
│  └─blog-vue                   //后台代码
│                  
└─后端
    └─robertBlog                //采用多模块架构  
        ├─admin                 //前端代码
        ├─blog                  //后端代码
        └─framework
```
